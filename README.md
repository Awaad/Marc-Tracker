# Activity Tracker (Proof of concept)

Proof of concept project to design a professional, extensible telemetry system that can measure
message delivery acknowledgements (where platforms support them) and visualize timing signals.

## Ethics / scope
This project is intended for self-owned accounts and explicit consent testing only.
Different platforms expose different receipt signals; the system models capabilities explicitly.

## Tech
- Backend: Python 3.12, FastAPI, WebSockets
- Frontend: React (Vite), TypeScript, Zustand
